# Clear the trash can  
```
rm -Rf /home/pi/.local/share/Trash/files/*
```
# Backup of user's data  
```
sudo su
mkdir /backup
chown pi:pi /backup
cd /backup; tar -cvf pi.tar /home/pi
```
